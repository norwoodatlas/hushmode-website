# Hush Mode ASO Playbook

## 1) App Store Metadata (Copy-Ready)

### Title options (max 30 chars)
- `Hush Mode: Focus & Screen Time`
- `Hush Mode: Focus Timer`
- `Hush Mode: Screen Time`

### Subtitle options (max 30 chars)
- `Pause before distracting apps`
- `Reduce scrolling, stay focused`
- `Build better phone habits`

### Keyword field options (max 100 chars, comma-separated, no spaces)
- `focus,productivity,screentime,habit,digitalwellbeing,appblocker,deepwork,mindful,adhd`
- `screentime,focusmode,productivity,discipline,phoneaddiction,habittracker,wellbeing,intention`

### Promotional text (max 170 chars)
- `Interrupt autopilot app checks with a short pause before distracting apps. Hush Mode helps you reclaim focus and reduce reactive screen time.`

## 2) Description Framework

### First 3 lines (most important for conversion)
- `Hush Mode helps you reduce screen time by adding a mindful pause before distracting apps.`
- `Break autopilot scrolling, stay intentional, and protect your attention.`
- `Built for iPhone with a behavior-first approach to focus.`

### Feature bullets
- `Pause before selected distracting apps`
- `Run short focus sessions`
- `See recap insights after each session`
- `Use gentle friction instead of hard lockouts`

### Trust section
- `Designed for privacy-conscious use with core data stored on-device where possible.`

## 3) Screenshot Sequence (5 Core Frames)

1. `Stop Autopilot Scrolling`
- Subtext: `Add a mindful pause before your most distracting apps.`

2. `Protect Your Focus Blocks`
- Subtext: `Use short sessions to stay intentional during work or study.`

3. `Choose with Intention`
- Subtext: `A quick breath beats another impulsive app check.`

4. `Track Your Progress`
- Subtext: `Review simple recap metrics after each session.`

5. `Build Better Phone Habits`
- Subtext: `Small daily wins compound into deeper focus.`

## 4) Icon + Visual Direction

- Keep icon simple and high-contrast at small sizes.
- Use a calm blue/white palette to signal focus and clarity.
- Avoid crowded icon details that blur in search results.

## 5) Ratings + Reviews Engine

- Trigger review request after a successful session recap, not on first launch.
- Ask after visible value moments (for example, 3 completed sessions).
- Respond to negative reviews with feature clarifications and fast fixes.

## 6) Release Notes Template

- `New: [feature that improves focus outcomes]`
- `Improved: [speed, reliability, UX]`
- `Fixed: [specific bug users noticed]`
- `Thanks for using Hush Mode. If it helps, a review supports the app.`

## 7) ASO Iteration Cadence

- Weekly: monitor impressions, page views, conversion rate, retention.
- Every 2 weeks: test subtitle + screenshot headline set.
- Monthly: refresh keyword field based on ranking and relevance.

## 8) Conversion Priorities

- First priority: screenshot headline clarity.
- Second priority: subtitle keyword intent match.
- Third priority: first 3 description lines.
